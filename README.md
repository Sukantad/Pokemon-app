## This is pokemon web app

### Tech Stack

- Next.js
- TailWindCss

### How to run in local

- First Clone this git repo
- npm i (install all dependencies package)
- npm run dev

### How to deploy on vercel

- First npm i -g vercel
- vercel and put your email id
- Deployed link- https://new-pokemon.vercel.app
